NAMELESS ENGINE SDK
About: NAMELESS ENGINE is an SDK written in C++ that supports 3D graphics with a C++ Interface. This tool is built with adaptability and simplicity as its core philosophies. Its modular architecture allows for simple and fast composition of a multitude of 3D Interactive virtual Environments while still maintaining a high level of control over what’s going on behind the scenes. Its purposes range from Video Game Development to Visualization of Artificial intelligence Models and other kinds of interactive systems were visualization is required. It should be used as a base for a software and extended upon Ad infinitum.
  • Type: Game Engine SDK
  • Developer: MateusBranco
  • Platform: PC-Windows
  • Written in: C++[Runtime]
  • Minimum System Requirements: OpenGL v4.5

    Boids
    About: Boids is an artificial life program, developed by Craig Reynolds in 1986, which simulates the flocking behaviour of birds. His paper on this topic was published in 1987 in the proceedings of the ACM SIGGRAPH conference. The name "boid" corresponds to a shortened version of "bird-oid object", which refers to a bird-like object.
